# Proyecto Auralis
Web oficial del Proyecto Auralis: noticias, herramientas y conocimiento económico

## ¿Qué es el Proyecto Auralis?.
Auralis es un proyecto abierto de conocimiento económico. 
Su objetivo es estudiar, sintetizar, ordenar y publicar contenido sobre economía, finanzas, mercados, etc. 
El proyecto se estructura sobre tres pilares principales:
- **Auralis News**: noticias y análisis breves sobre acontecimientos con impacto económico, de mercado, etc.
- **Auralis Knoledge**: artículos, documentos y materiales introductorios sobre economía, finanzas y sus campos relacionados.
- **Auralis Programs**: herramientas y programas de código abierto orientados al aprendizaje, cálculo y análisis.

## Sobre este repositorio. 
Este repositorio contiene la web principal del proyecto. Aquí se desarrollan y publican:
- La portada del proyecto.
- Las páginas institucionales.
- La estructura de navegación.
- Los contenidos de **Auralis News**.
- Los materiales de **Auralis Knoledge**.
- Y una pirmera integración con **Auralis Programs**.

## Estado del proyecto. 
Auralis se encuentra en fase de desarrollo incial. La priemra versión del sitio servirá como piloto público del proyecto y contendrá:
- Una página principal.
- Una base institucional mínima.
- Los primeros contenidos editoriales.
- Los primeros programas accesibles desde la web.
